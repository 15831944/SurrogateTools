The July 2005 version of the Spatial Allocator includes the following new features:

  • The original mims_spatial.exe program has been divided into:
    • Allocator.exe, performs spatial allocation, overlay, and filtering of shapefiles
    • creating surrogates based on filtered shapefiles - using this feature, you can apply a filter to a shapefile and create a surrogate on the resulting data all in one run of the spatial allocator
  • srgcreate.exe, generates surrogates
    • surrogates based on a mathematical functions of attributes in a shapefile
  • srgmerge.exe, merges surrogates, creates surrogates that are weighted functions of existing surrogates or create new surrogates by applying "gap filling" to existing surrogates
  • Biogenic emissions program

 

The January 2005 version of the Spatial Allocator has the following new features:

  • filtering of shapefiles - using this feature, you can create a new shapefile that is a subset of a larger shapefile according to criteria you specify for the attributes of the shapefile
  • creating surrogates based on filtered shapefiles - using this feature, you can apply a filter to a shapefile and create a surrogate on the resulting data all in one run of the spatial allocator
  • generate surrogates based on a mathematical functions of attributes in a shapefile
  • create new surrogates that are weighted functions of existing surrogates using the new srgmerge program
  • create new surrogates by applying "gap filling" to existing surrogates by using the new srgmerge program
