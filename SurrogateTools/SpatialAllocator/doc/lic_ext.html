External Libraries

The programs that make up the MIMS Spatial Allocator were created using multiple packages:

  • PROJ.4 - see http://proj.maptools.org
  • Shapelib - see http://shapelib.maptools.org
  • General Polygon Clipper - see http://www.cs.man.ac.uk/aig/staff/alan/software
  • GNU regex library - see http://www.gnu.org/directory/regex.html

The versions used in the construction of this program can be found in the external subdirectory.
